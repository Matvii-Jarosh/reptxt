# README

## Package Name
**reptxt**

## Version
1.0

## Description
reptxt is a utility designed to efficiently replace specified text within input streams and files. It simplifies the process of editing and manipulating text, making it an ideal tool for developers and users working with large volumes of textual data.

## Installation
Detailed installation instructions can be found in the [INSTALL](INSTALL) file. Please review it before getting started.

- **doc/** — contains the source documentation in .texi format.
- **src/** — includes the source code of the utility.
